# Robb world

Base de blog pessoal colorido e criativo, organizada para crescer sem bagunca.

## Como usar

Abra `index.html` no navegador. A pagina inicial fica na raiz do projeto, e as
postagens ficam dentro da pasta `posts`.

## Estrutura

- `index.html`: pagina inicial do blog
- `pages/jogos.html`: pagina especial de jogos favoritos
- `pages/musicas.html`: pagina especial de musicas favoritas
- `posts/opinioes/`: postagens de opiniao
- `css/styles.css`: visual, responsividade e cores
- `js/blog.js`: busca e filtro por categoria
- `assets/images/covers/`: imagens de capa das postagens

## Redes sociais

Os icones de redes sociais estao no header e no rodape da home. Troque os
`href="#"` pelos links reais do Instagram, GitHub, LinkedIn, YouTube, Spotify e
Twitter.

## Criando uma postagem nova

Duplique uma postagem existente dentro de `posts`, renomeie com letras minusculas
e hifens, edite o conteudo e adicione um card na home apontando para o arquivo
novo.
